## Assignment 5 - Unit, Mocking and Integration Testing

## Project Overview

This project simulates two retail systems: BarnesAndNoble and Amazon. It includes unit tests using mocks
and integration tests to verify component interactions, including database operations and price rules.
JUnit 5, Checkstyle, and JaCoCo ensure functionality, code quality, and high test coverage.
GitHub Actions automatically runs tests, static analysis, and coverage reports on every push.
